#PORTFOLIO

[앱 명칭] : 털뭉치들  

[프로젝트 기간] 2021.02.01 ~ 2021.03.05  

[프로젝트 설명] 내가 키우는 귀여운 강아지, 집 밖에 나가가지 못해 자랑하지 못했던 고양이 등 반려동물을 자랑해보는 커뮤니티

[프로젝트 인원] 1명  

[개발환경] 안드로이드 스튜디오, git  

[사용기술]
- Kotlin, Java
- 서버: Firebase
- 주요 라이브러리 및 API : Room, Retrofit2, Glide, Anko, Koin, AAC, 카카오 링크 공유 Api
- 디자인 패턴 : ViewModel + LiveData를 이용한 MVVM 패턴 
- 데이터 베이스 : SQLlite
